version info

5 min read feb 2026

overview

stremio is available across many many platforms but the version you need depends on your device and its architecture. i found it confusing which one to pick (particularly on android tv) when i began using stremio so i made this guide.

note: all downloads can be found on the official stremio downloads page. this guide only explains which version to select.

windows

stremio offers two builds for windows. most users will need x64.

  • x64 - for standard intel/amd processors. this is what the most windows PCs will use.
  • arm - for newer arm-based windows devices such as the surface pro x, surface pro 9 (5g), thinkpad x13s, and other snapdragon-powered laptops.
how to check: press Win + I → system → about. look at "system type" - it will say "x64-based processor" or "arm-based processor".

macos

there are two builds depending on your mac's chip.

  • arm (apple silicon) - for macs with an M1, M2, M3, M4 chip or later.
  • intel - for older macs with an intel processor (pre-late 2020).
how to check: click the apple menu → about this mac. if the chip line says "apple M…" you need arm. if it says "intel" you need intel.

linux

all linux builds are 64-bit only. choose whichever format best suits your distro and package manager.

  • flatpak - works on most distros via flathub. easiest option if you're unsure.
  • .deb - for debian-based distros (ubuntu, pop!_os, linux mint, etc.).
  • aur - for arch-based distros (arch, endeavouros, manjaro, etc.). install via your preferred aur helper.
  • source code - build from source if your distro isn't covered above or you need a custom build.

android tv

stremio has four android tv builds. most modern android tv devices use arm64.

  • arm - older 32-bit arm devices (also the chromecast with google tv and onn 4k models because they run a 32-bit android os).
  • arm64 - most modern android tv boxes and smart tvs (nvidia shield, fire stick 4k max etc).
  • x86 - rare 32-bit intel-based android tv devices.
  • x86_64 - rare 64-bit intel-based android tv devices.
tip: if you're unsure, try arm64 first - it covers the vast majority of modern android tv hardware. if it doesn't install, try arm.

android mobile

same architecture options as android tv but packaged for phones and tablets.

  • arm - older 32-bit phones.
  • arm64 - most phones released after ~2017.
  • x86 - very rare, mostly older chromebooks or emulators.
  • x86_64 - very rare, some chromebooks or emulators.
how to check: go to settings → about phone and look for "CPU" or "processor". alternatively, install CPU-Z and check the "abi" field.

samsung tv

stremio is available on samsung smart tvs running tizen OS from 2019 onwards.

  • download directly from the samsung app store on your tv.
  • tvs older than 2019 are not supported.
how to check your model year: go to settings → support → about this tv. the model number contains the year - e.g. UN55TU8000 = 2020 (TU series).

lg tv

stremio is available on lg smart tvs running webOS from 2020 onwards.

  • download directly from the lg content store on your tv.
  • tvs older than 2020 are not supported.

sony tv (android/google tv)

sony tvs that run android tv or google tv can install stremio directly from the google play store or via sideloading the android tv apk.

  • only sony tvs running android tv / google tv are supported.
  • older sony tvs running their proprietary os cannot run stremio.
how to check: go to settings → about. if you see "android tv" or "google tv" in the software info, you're good.

philips tv

philips sells two types of smart tvs with different operating systems.

  • android tv / google tv models - full stremio support. install from the play store or sideload the android tv apk.
  • titan os models - limited stremio version only. this version has reduced functionality compared to the full android build (such as no support for torrents).
how to check: go to settings → about. if you see "android" or "google tv" you have the full version. if you see "titan os" you will only get the limited build.

hisense tv

hisense also ships tvs with two different operating systems.

  • android tv / google tv models - full stremio support. install from the play store or sideload the android tv apk.
  • vidaa models - limited stremio version only. this version has reduced functionality compared to the full android build (such as no support for torrents).
how to check: go to settings → system → about. if you see "android" or "google tv" you have the full version. if you see "vidaa" you will only get the limited build.

steam deck

the steam deck runs a modified arch linux (steamos), so use the flatpak version.

  • switch to desktop mode (hold the power button → switch to desktop).
  • open the "discover" app store and search for stremio, or install via the terminal with flatpak install flathub com.stremio.Stremio.

ios

there is no native stremio app for ios without sideloading as of feb 2026 (due to apple removing the stremio lite app from both ios and appletv stores). the only option is a limited web version.

  • open web.stremio.com in safari.
  • tap the 3 dots in the bottom right → press the share button → view more → "add to home screen" to create a web app shortcut (ensure 'open as web app' is enabled).
  • this version cannot stream torrents - it only works with http streams, which means it is only useful if you use a debrid service.

meta quest

stremio is available on meta quest headsets in two modes.

  • vr version - immersive virtual cinema experience for watching content in vr.
  • 2d version - flat panel mode, behaves like a standard android app inside the headset.

pico

stremio is available on pico vr headsets in two modes.

  • vr version - immersive virtual cinema experience for watching content in vr.
  • 2d version - flat panel mode, behaves like a standard android app inside the headset.

raspberry pi 4

the raspberry pi 4 can run stremio os - a dedicated operating system that turns the pi into a stremio set-top box.

  • download the raspberry pi 4 stremio os image from the stremio downloads page.
  • flash it to an sd card using raspberry pi imager or balena etcher.
  • this is a different image from the pi 5 version - make sure you download the correct one.

raspberry pi 5

the raspberry pi 5 also supports stremio os, but uses a separate image optimised for its hardware.

  • download the raspberry pi 5 stremio os image from the stremio downloads page.
  • flash it to an sd card using raspberry pi imager or balena etcher.
  • do not use the pi 4 image on a pi 5 or vice versa - they are not interchangeable.

stremio launch on boot app

a small companion app that automatically launches stremio as soon as your device boots up. useful for dedicated stremio setups like a tv box or projector.

  • available on android devices only.
  • once installed and given proper permissions, stremio will start automatically every time the device powers on.
