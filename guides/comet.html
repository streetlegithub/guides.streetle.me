comet

7 min read feb 2026

overview

comet is a popular stremio addon. it fetches torrents from various sources and allows you to stream them directly within stremio.

it differs from torrentio in that it can be self hosted and therefore has many instances ran by different users in-case of downtime.

before you begin

you may need a debrid service depending on the country you live in, it is also generally recommended for stable streams that don't require torrenting.

follow the debrid guide to get started with debrid services.

note: all (good) debrid services cost money.

configuration steps

1

navigate to the comet configuration page.

2

optional step - select your debrid provider and enter your API key.

3

configure quality filters and sorting options.

for example if your device is only capable of 1080p, remove the 4k UHD and QHD option in the resolution filter.

4

install the configured addon to stremio.

settings overview

comet exposes a variety of options in its settings panel (more than torrentio). below is a brief description of each setting.

resolution

select all resolutions you wish to include in the results. unselecting "unknown" is not recommended for niche content that may not have a resolution tag in its torrent file.
unselect any resolution you know your device does not support

max results per resolution

set the maximum number of results to show for each resolution. this is useful for reducing clutter in the results list.

debrid services

add the debrid service you wish to use (real-debrid, torbox, etc.) or ignore this section if you do not use one. a debrid service will be more stable as it doesn't require you to download via torrenting.

for more information on debrid services, see the debrid guide.

api key

enter your debrid account api key. this field is required when a provider is selected; otherwise comet will be unable to use debrid services.

enable torrent mode

enable this option if you want to use torrent mode instead of debrid mode. this is useful if you don't have a debrid service or want to use torrenting when your debrid service has not cached the torrent yet.

note: be careful when using torrent mode as it will expose your IP address to third parties. if you live in a country where torrenting rules are enforced, make sure to use a vpn or only torrent content that is legal in your country.

deduplicate streams

enable this option to remove duplicate streams from the results list. this is useful for reducing clutter in the results list.

scrape debrid account torrents

enable this option to scrape torrents from your debrid account. this is useful if you want to see all torrents in your debrid account in the results list.
