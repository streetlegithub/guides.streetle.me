getting started with stremio

3 min read feb 2026

overview

learn how to install stremio and create an account

at the end of this guide you will have stremio installed on your device, an account set up and ready to begin streaming content.

installation steps

1

navigate to stremio.com/downloads.

note: i have a dark mode extension so it may look different for you, but the layout should be the same.
2

download and install stremio for your operating system (windows, mac, linux, android tv, android).

if you're unsure of which version to download (arm64, x86, arm64, x86_64), see the version info guide.

3

launch stremio and create an account or sign in if you already have one.

note: signing in with facebook will make your profile picture your facebook profile picture permanently.

signing in with apple may have issues due to a revoked developer account.

alternative (no install)

navigate to web.stremio.com and create an account or sign in if you already have one.

note: the web version of stremio is very limited, as it cannot stream torrents, this is irrelevant if you use a debrid service

finished

you should now have stremio installed and an account set up. you're ready to install basic addons.

if you just want simple streaming, follow the basic addons guide.

if you are an advanced user, you may want to explore aiostreams.
